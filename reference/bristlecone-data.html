Bristlecone.Data Namespace

Contains functions that enable loading and saving of core Bristlecone types to file storage. Bristlecone loads and saves CSV data for both individual model results (MLEs, traces, predicted vs observed series), and for ensemble-level statistics (model selection / weights).

Config

Specifies how file paths are constructed for different data types.

MLE

Functions of loading and saving Maximum Likelihood Estimates from model fits

NStepAhead 

Loads and saves one-step-ahead and n-step-ahead prediction
 results into and out of simple CSV files

Series
